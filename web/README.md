# 壮乡乐脉 Web版本

这是壮乡乐脉小程序的Web版本，可以通过浏览器直接访问。

## 如何部署

### 方法1：使用GitHub Pages
1. 将整个 `web` 目录上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择 `web` 目录作为发布源
4. 等待几分钟，GitHub Pages会生成访问链接

### 方法2：使用其他静态网站托管服务
- Netlify
- Vercel
- 阿里云OSS
- 腾讯云COS

## 如何访问

部署完成后，通过以下方式访问：
1. GitHub Pages: `https://username.github.io/repository/web`
2. 其他托管服务：根据服务提供商的说明获取访问链接

## 功能说明

- **首页**：导航到各个功能页面
- **创作**：生成音乐，支持选择音乐风格、长度、速度和情感
- **展示**：查看音乐作品列表，支持按分类筛选
- **互动**：发布和查看评论
- **我的**：查看个人信息和创作历史
- **作品详情**：查看作品详情，支持播放音乐、点赞、收藏和评论

## 技术栈

- HTML5
- CSS3
- JavaScript

## 注意事项

- 音频文件需要确保路径正确
- 部分功能为模拟实现，实际使用需要后端支持
- 响应式设计，适配不同屏幕尺寸
